# GitHub Pages publish checklist

Upload only the files from this folder to a public GitHub repository:

- `index.html`
- `glass-bathroom-model.glb`
- `.nojekyll`

Then open the repository on GitHub:

1. Go to `Settings`.
2. Open `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Click `Save`.

GitHub will show the public site URL after the first deployment finishes.
